# B2G 미세먼지 분석 프로젝트

## 데이터
- 위치: ./기간별_일평균_대기환경_정보_2024년.csv + ./기간별_일평균_대기환경_정보_2025년.csv
- 인코딩: CP949 (EUC-KR 호환)
- 출처: 서울 열린데이터광장 OA-2218 (일별 평균 대기환경)
- 시간 범위: 2024-01-01 ~ 2025-12-31 (2년치, YoY 비교)
- 측정소: 서울 25개 자치구
- 컬럼명 통일: 2024 "미세먼지/초미세먼지" → 2025 "미세먼지농도/초미세먼지농도" 와 일치하도록 분석 시 통일

## 분석 규칙
- 모든 결과는 한글로 출력
- 차트 한글 폰트 깨지지 않게 처리 (Mac: AppleGothic, Windows: Malgun Gothic)
- seaborn 사용 시: sns.set_theme() 먼저 호출 → 그 다음 폰트 재지정
- 차트는 PNG로 ./outputs/ 폴더에 저장
- 측정일시는 datetime으로 파싱
- 시간 범위 답할 때 항상 연도 명시 (또는 두 해 통합)

## 출력
- 코드 설명은 한 줄 요약
- 결과 인사이트는 핵심 1~2 문장
