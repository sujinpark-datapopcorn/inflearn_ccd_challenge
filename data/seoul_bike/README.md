# 서울 따릉이 — 일별 대여 건수

> **원칙**: 출처에서 직접 다운로드. 다운로드 못 한 분만 이 폴더의 백업본 사용.

## 출처
- 서울 열린데이터광장 — 공공자전거 이용정보(시간대별)
- https://data.seoul.go.kr/dataList/OA-15182/F/1/datasetView.do
- 검색 키워드: "따릉이 대여" / "공공자전거 일별"
- 회원가입 불필요 · CSV 즉시 다운로드

## 다운로드 방법
1. 위 URL 접속 또는 광장에서 "따릉이 일별" 검색
2. 상단 "파일" 탭 → 원하는 연도 CSV 다운로드
3. 본인 작업 폴더로 이동

## 데이터 정보 (백업본 기준)
- 컬럼 2개: 대여일자, 대여건수
- 인코딩: CP949
