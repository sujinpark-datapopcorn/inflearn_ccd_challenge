# 서울 따릉이 일별 이용 데이터

## 출처
- **서울 열린데이터광장 OA-15246** — 서울시 공공자전거 따릉이 이용정보(일별)
- https://data.seoul.go.kr/dataList/OA-15246/F/1/datasetView.do
- 회원가입 불필요 · 월별 CSV 즉시 다운로드
- 페이지에 "종료된 서비스" 표기는 있지만 다운로드 가능

## raw 형태
- **7컬럼**: `대여일자, 대여소, 정기권유무, 성별, 연령대, 운동량, 이용건수`
- 파일 패턴: `서울특별시 공공자전거 이용정보(일별)_YYMM.csv`
- 월별 50~240 MB (큼)
- 인코딩: CP949

## 이 폴더 안

### `raw_2024/` — 광장 OA-15246 원본 (2024년 12개월)
강의 레포 백업본으로 보존. raw 그대로 받아 분석할 때 사용.

### `seoul_bike_daily_2024.csv` — 사전 집계본 (시간 절약용)
강의에서 시간 아끼려고 raw 12개월에서 **일별 합계로 미리 집계**한 가공본.
- 컬럼 2개: `대여일자, 대여건수` (성별·연령대·정기권 등은 제거)
- 366행 (2024년)

## 받기 + 가공 흐름

### 빠른 길 — 사전 집계본 그대로
- `seoul_bike_daily_2024.csv` 다운로드 → 일별 분석에 즉시 사용

### 정공법 — raw에서 직접 집계
1. 광장에서 OA-15246 12개월 raw 다운로드 (또는 `raw_2024/` 백업본)
2. Claude Code에 한 마디:
   ```
   서울특별시 공공자전거 이용정보(일별)_2401.csv ~ _2412.csv 12개 파일 통합해서
   일별 이용건수 합계 구해줘. 모두 CP949 인코딩이야.
   ```
3. 또는 성별·연령대 등 풍부 컬럼 활용해서 더 깊은 분석:
   ```
   일별 + 성별 + 연령대 분포로 집계해서, 미세먼지 등급별 어떤 그룹이
   환경에 가장 민감한지 분석해줘.
   ```

## 활용 예시 (Day 2)
- PM2.5 등급별 따릉이 일별 대여 (사전 집계본으로 충분)
- raw 활용 시: 성별·연령대별 환경 민감도 비교
- 황사 시즌 vs 비황사 시즌 따릉이 이용 패턴
