# 검색량/언급량 — Day 2 결합용

강의 시작 전 강사가 사전 배포 예정 (`buzz_keywords.csv`).

## 출처
- 네이버 데이터랩 — 키워드별 일별 검색 비율
- 구글 트렌드 — 검색 빈도 인덱스 (0~100 정규화)

## 수집 키워드 (예정)
- "미세먼지", "초미세먼지", "황사"
- "마스크", "공기청정기"
- (추가) "환기", "필터"

## 컬럼 (예정)
- `date` (YYYY-MM-DD)
- `keyword`
- `search_index` (0~100 정규화)
- `source` (`naver` / `google`)

## 활용 예시
- PM2.5 농도 vs "미세먼지" 검색량 시계열 + 상관 lag
- 사람들이 PM 몇 ㎍/㎥부터 검색하는지 임계값 추정
- "마스크"·"공기청정기" 검색 급등 vs PM 실측 — 어느 게 며칠 빠른가
