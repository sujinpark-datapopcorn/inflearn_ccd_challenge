# 검색량/언급량 — Day 2 결합용

> **원칙**: 출처에서 직접 다운로드. 어려운 분은 인프런에서 강사 사전 배포본을 받기.

## 출처
- 네이버 데이터랩 — 키워드별 일별 검색 비율 (https://datalab.naver.com)
- 구글 트렌드 — 검색 빈도 인덱스 0~100 (https://trends.google.com)

## 다운로드 방법
- 네이버 데이터랩: 검색어 트렌드 → 키워드 입력 → 기간 설정 → 다운로드 (CSV)
- 구글 트렌드: 키워드 검색 → 우상단 다운로드 아이콘 (CSV)

## 권장 키워드
- "미세먼지", "초미세먼지", "황사"
- "마스크", "공기청정기"
- (확장) "환기", "필터"

## 컬럼 (권장)
- `date` (YYYY-MM-DD)
- `keyword`
- `search_index` (0~100 정규화)
- `source` (`naver` / `google`)

## 활용 예시 (Day 2)
- PM2.5 농도 vs "미세먼지" 검색량 시계열 + 상관 lag
- 사람들이 PM 몇 ㎍/㎥부터 검색하는지 임계값 추정
- "마스크"·"공기청정기" 검색 급등 vs PM 실측 — 어느 게 며칠 빠른가
