# 기상청 ASOS 서울 일자료 — Day 2 결합용

> **원칙**: 출처에서 직접 다운로드. 어려운 분은 인프런에서 강사 사전 배포본을 받기.

## 출처
- 기상자료개방포털 — ASOS 서울 일자료
- https://data.kma.go.kr → 기후통계분석 → 기온/강수/바람 등 일자료
- 기상청 회원가입 후 CSV 다운로드 가능

## 다운로드 방법
1. 기상자료개방포털 접속 → 회원가입 (무료)
2. "기후통계분석" → "통계분석" → 지점 "서울(108)", 기간 2024-01-01 ~ 2025-12-31
3. CSV로 저장

## 수집 컬럼 (권장)
- `date` (YYYY-MM-DD)
- 평균기온, 최저기온, 최고기온
- 일강수량
- 평균풍속, 최대풍속, 풍향
- 평균상대습도

## 인코딩
- UTF-8 (강의 흐름 통일 — 받은 후 변환 가능)

## 활용 예시 (Day 2)
- PM2.5와 풍속·강수·기온의 상관관계
- 황사 시즌 풍향 분포
- "맑고 풍속 약함 + 한파" 정체 시나리오 검증
