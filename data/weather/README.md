# 기상청 ASOS 서울 일자료 — Day 2 결합용

강의 시작 전 강사가 사전 배포 예정 (`weather_seoul_daily.csv`).

## 출처
- 기상자료개방포털 ASOS (Automated Surface Observing System) 서울 지점 일자료
- 기상자료개방포털: https://data.kma.go.kr

## 수집 컬럼 (예정)
- `date` (날짜, YYYY-MM-DD)
- 평균기온, 최저기온, 최고기온
- 일강수량
- 평균풍속, 최대풍속, 풍향
- 평균상대습도

## 인코딩
- UTF-8 (강의 흐름 통일)

## 활용 예시
- PM2.5와 풍속·강수·기온의 상관관계
- 황사 시즌 풍향 분포
- "맑고 풍속 약함 + 한파" 정체 시나리오 검증
