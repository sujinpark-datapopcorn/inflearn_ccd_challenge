# 기상청 ASOS 서울 일자료 — Day 2 결합용

> 기상청 직접 다운로드는 **회원가입 + 요소 선택이 까다로워**, 이 폴더의 백업본을 그대로 쓰는 걸 권장.
> 본인 분석에서 다른 요소가 필요하면 출처에서 직접 받기.

## 백업본 (이 폴더)
- `OBS_ASOS_DD_20260510160758.csv`
- 인코딩: **CP949**
- 731행 · 2024-01-01 ~ 2025-12-31
- 컬럼:
  - 지점, 지점명, 일시
  - 평균기온(°C), 최저기온(°C), 최고기온(°C)
  - 일강수량(mm)
  - 평균 풍속(m/s), 최다풍향(16방위)
  - 평균 상대습도(%)

## 직접 다운로드 (선택)
- 기상자료개방포털: https://data.kma.go.kr
- 회원가입 → 데이터 → 기상관측 → 지상관측 → ASOS → 일자료
- 지점: 서울(108) / 기간: 2024-01-01 ~ 2025-12-31
- 요소 선택: 위 7개 (평균기온·최저·최고·일강수량·평균풍속·최다풍향·평균습도)

## 활용 예시 (Day 2)
- PM2.5와 풍속·강수·기온의 상관관계 (풍속 -0.30 가장 강한 음의 상관)
- 황사 시즌 풍향 분포
- 정체 시나리오: "맑고 풍속 약함 + 한파"
