# 서울 지하철 — 일별 총 승하차

> **원칙**: 출처에서 직접 다운로드. 다운로드 못 한 분만 이 폴더의 백업본 사용.

## 출처
- 서울 열린데이터광장 — 지하철 이용 현황 (일별)
- https://data.seoul.go.kr 에서 "지하철 일별 승하차" 검색
- 회원가입 불필요 · CSV 즉시 다운로드

## 다운로드 방법
1. 광장에서 "지하철 일별 승하차" 검색
2. 상단 "파일" 탭 → 원하는 연도 CSV 다운로드
3. 본인 작업 폴더로 이동

## 데이터 정보 (백업본 기준)
