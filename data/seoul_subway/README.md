# 서울 지하철 — 일별 시간대별 승하차

> **원칙**: 출처에서 직접 다운로드. 다운로드 못 한 분만 이 폴더의 백업본 사용.

## 출처
- **공공데이터포털 (data.go.kr)** ★ 서울 열린데이터광장 아님
- 정식 명칭: 서울교통공사_역별 일별 시간대별 승하차인원 정보
- https://www.data.go.kr/data/15048032/fileData.do
- 회원가입 불필요 · CSV 즉시 다운로드

## 다운로드 방법
1. 위 URL 접속
2. 페이지 상단 "파일데이터" 탭 → 원하는 연도 CSV 다운로드
3. 본인 작업 폴더로 이동
4. 받아지는 파일명 예: `서울교통공사_역별 일별 시간대별 승하차인원 정보_YYYYMMDD.csv`

## 데이터 정보 (raw vs 백업본)
- **raw**: 199,424행 × 26컬럼 (수송일자·호선·역·승하차구분·24시간대 wide format)
- 인코딩: CP949
- **백업본** (`seoul_subway_daily_total_2024.csv`): raw를 일별 총승하차 합계로 가공
  - 컬럼: 수송일자, 총승하차인원, 고유역수, 행수
  - 366일 (2024년 1년)
