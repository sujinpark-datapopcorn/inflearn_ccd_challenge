# 서울 미세먼지 — 일별 평균 대기환경 정보

> **원칙**: 출처에서 직접 다운로드. 다운로드 못 한 분만 이 폴더의 백업본 사용.

## 출처
- 서울 열린데이터광장 — OA-2218 일별 평균 대기환경 정보
- https://data.seoul.go.kr/dataList/OA-2218/S/1/datasetView.do
- 회원가입 불필요 · CSV 즉시 다운로드

## 다운로드 방법
1. 위 URL 접속
2. 상단 "파일" 탭 클릭
3. **2024년 자료 + 2025년 자료** 각각 CSV 다운로드
4. 본인 작업 폴더로 이동 (파일명은 받은 그대로 두면 됨)

## 데이터 정보
- 행 수: 18,275행 (2024: 9,150 + 2025: 9,125)
- 컬럼 9개: 측정일시, 권역명, 측정소명, 이산화질소(ppm), 오존(ppm), 일산화탄소(ppm), 아황산가스(ppm), 미세먼지(㎍/㎥), 초미세먼지(㎍/㎥)
- 인코딩: **CP949** (EUC-KR 호환)
- 측정소: 서울 25개 자치구
- 컬럼명 통일 안내: 2024 "미세먼지/초미세먼지" → 2025 "미세먼지농도/초미세먼지농도" 와 일치하도록 분석 시 통일
